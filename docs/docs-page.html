Overview

This free, open-source API consists of a series of technical analysis data for understanding the financial markets. It provides optimal trading zones inspired from, but not limited to, the T-Cup trading strategy and also leveraged the Theory of True Support and True Resistance in Technical Analysis. These zones are defined as the closest, the last and the closest-most-recent zones. The API returns them in this format for numerous trading pairs, stocks commodites, across numbeous time frames.

Trading Theory

Markets follow one or many Models at any given time, and on any given timeframe. In adhering to this model, there are key zones that the market acts on regardless of the model. Some may deem these zones, physchological, fundamental; though, through rigorous backtesting, we found these overlapping zones, and have names these to be Augustus Zones. An example of this is shown on the T-Cup Model below - originally called the Cup and handle Model by William O'Neil in 1988.

Augustus' Zones

Augustus' Trading Zones, what this API provides, are three main zones: namely the Closest Zone (CZ), Closest Most Recent Zone (CMRZ) and Last Zone (LZ). These are all true support and true resistance zones, and therefore before defining the Augustus Zones, it is important to define what True Support and True Resistance is.

True Support & Resistance

Support and Resistance have been defined by mathematicians as the first derivative of a given trading graph, ie the peaks and troughs, where the derivatives are 0. While this is indeed accurate, it does not take into account the importance of price-action; which would further define support and resistance solely on three indicators: movement, turbulence and reversal/confirmation.

Augustus' Zones

Zones found in this image are further defined below

Closest Zone (CZ)

Defined as the closest zone relative to where the market is now

Closest Most Recent Zone (CMZ)

Defined as the closest most recent zone relative to where the market is now

Last Zone (LZ)

Should not be confused with the CMRZ , but is the last cronoligical zone to have been respected

Strength Index

A value used to give a zone a weight, and is a direct replica of the number of times the zone has been respected

Note

The Last Zone (LZ) should not be confused with the Closest Most Recent (CMRZ), the latter is sorted by proximity and then cronology.

  • If you are a spreadsheet user (e.g., Excel or Google Sheets), please check out our dedicated spreadsheet add-ons.

  • Looking for inspirations? Feel free to reach out: we have quantitative investing, machine learning, web development, and much more, all curated and delivered to you by industry experts.



Are there usage/frequency limits for the API service?

We are pleased to provide free stock API service for our global community of users for up to 5 API requests per minute and 500 requests per day. If you would like to target a larger API call volume, please visit premium membership.


You support both raw and adjusted intraday/daily/weekly/monthly time series. What adjustment method are you using?

We use the adjustment methodology from the Center for Research in Security Prices (CRSP), an industry standard.


I have built a library for AugustusAI with a specific programming language. May I open-source it on GitHub?

Certainly - we truly appreciate the help and support from the community to make AugustusAI even more accessible and developer-friendly. However, we ask that your language-specific library/wrapper preserves the content of our JSON/CSV responses in both success and error cases. We consider it a top priority that our users get the original debugging and troubleshooting information from Alpha Vantage.


I would like to improve an existing API or propose a new technical feature to AugustusAI. What should I do?

We are obsessed with delivering the best user experience with the highest technical expertise. Please share your feature requests with us at support@alphavantage.co.
